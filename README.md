# Bellcorp Event Management Application

A full-stack event management platform built with React, Node.js, and SQLite.

## Features
- User Authentication (JWT)
- Event Discovery (Search & Filter)
- Event Registration & Cancellation
- User Dashboard (Upcoming/Past events)

## Getting Started

### Backend
```bash
cd server
npm install
node seed.js
node server.js
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Axios, Lucide
- **Backend**: Node.js, Express, Sequelize, SQLite, JWT
